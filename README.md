# アプリの概要
「ゆるトレメイト (AI Personal Weight Trainer)」 は、日々の体重をシンプルに記録しながら、AIがあなたの専属トレーナーとしてサポートしてくれる体重管理アプリです。
無理なく楽しくダイエットやボディメイクを続けられるように、体重の変化をグラフで可視化するだけでなく、ユーザーの「目的」や「環境」に合わせた最適なトレーニングメニューをAIが日替わりで提案してくれます。さらに、目標体重までの進捗に合わせて温かい応援メッセージを送ってくれるため、モチベーションを維持しながら記録・トレーニングを続けることができます。
# 技術スタック
**このアプリは、最新のモダンなWebフロントエンド技術を取り入れて構築されています。:**
 * **コア技術:** (フロントエンド)React 19、TypeScript:
 * **ビルドツール:** Vite (高速な開発・ビルド環境)
 * **スタイリング:** Tailwind CSS (ユーティリティクラスによるモダンな装飾)
 * **アニメーション:** Motion (旧 Framer Motion / スムーズなUIトランジション)
 * **UIアイコン:** Lucide React
 * **グラフ描画:** Recharts (体重記録の可視化)
 * **AI・バックエンド機能:** Google GenAI SDK (@google/genes）
ユーザーごとのパーソナライズされたトレーニングメニューの考案やメッセージの生成に最先端の Gemini モデル を活用しています。
# 使い方
アプリの使い方はとてもシンプルです。以下のステップで毎日の記録とトレーニングを進められます。
1. プロフィールの設定:* まずはアプリ内の設定画面で、あなたの情報（身長、年齢、性別、目標体重）を入力します。
2. 「目的」と「環境」の選択:* このアプリで何を達成したいか（「減量したい」「筋肉をつけたい」「維持したい」「記録だけしたい」）を選び、普段のトレーニング環境（「自宅」「ジム」「特になし」）を選択します。
3. AIおすすめトレーニングの確認:* 目的（※「記録だけしたい」以外）を選択すると、あなたのプロフィールや選んだ環境に合わせて、Gemini AIが「今日の最適なトレーニングメニュー（種目や回数など）」と「モチベーションが上がる一言」を自動で考えてトップ画面に提案してくれます。
4.日々の体重記録:* 毎日体重を測って記録（ログ追加）します。記録されたデータはダッシュボードのグラフに追加され、これまでの推移が一目でわかります。
5. 応援メッセージの受け取り:* 記録のたびに、画面の上部に「目標まであと〇〇kg！」という進捗と共に、AIからの応援メッセージ（例：「頑張って！目標までもう少しだよ✨」「素晴らしいキープだね！🎉」）が表示され、あなたの頑張りを後押ししてくれます！
